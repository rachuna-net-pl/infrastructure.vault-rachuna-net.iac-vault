## Conventions

### File Format
- All files: `.tf.json` (JSON syntax)
- Disabled files: `.tf.json.txt` (ignored by OpenTofu)
- **Dotfiles ignored** — `.*.tf.json` is NOT loaded! Use `locals.tf.json`, not `.locals.tf.json`

### Branch Naming
- `feat/user-<username>` — new user
- `feat/<feature>` — features
- `fix/<bug>` — fixes
- `docs/<topic>` — documentation
- Conventional Commits: `feat:`, `fix:`, `docs:`

### Module Sources
- External: `git::https://gitlab.com/...?ref=v1.0.0` (always use tags, not branches)
- Local: `"./path/"`

### Secrets
- **NEVER** secrets in repo (gitlab.com is public)
- Passwords: generated by `random_password`, stored in KV v2
- Root token: given manually, never written
