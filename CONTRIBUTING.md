# Contributions

Jeśli masz pomysły na ulepszenia, zgłoś problemy, rozwidl repozytorium lub utwórz Merge Request. Wszystkie wkłady są mile widziane!

## Jak pomóc?
1. Zgłoś issue z opisem problemu lub propozycją
2. Stwórz branch z prefixem `feat/`, `fix/`, `docs/`
3. Wypchnij zmiany i utwórz Merge Request
4. MR będzie zreview'owane przez @mrachuna
